# EBIMU Qt
Project that gets the value of 'ebimu24gv3' through serial and calculate yaw value.
Based on serial code from E2box.

==========================================
Environment
Qt Ver: 6.4.3
Qt Creator Ver: 10.0.1
Compiler: mingw 11.2.0-64bit
OS: Windows 11
